## 遗传算法

仅对于使用遗传算法解决N皇后问题，我个人的理解是可以看作一个经过多初始值和不定步长的局部搜索问题。所以参数的设定的思路和局部搜索一致，种群的设置不能过大。

本次作业运行结果如下
| N           | 算法运行时间 |
| ----------- | -----------: |  
| 8      | 0s       |   
| 16      | 0.95 s       |
| 32      | 1.57 s       |
| 64      | 0.46 s       |
| 128      | 242.05 s       |
| 256      | 89.62 s       |
| 512      | 2444.74 s       |

从运行结果看，遗传算法的收敛和复杂相关，但是和随机获取的初始值也有比较大的关系
```
g++ -std=c++11 genetic_algorithm.cpp ; ./a.out
```
